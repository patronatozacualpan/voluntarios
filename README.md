# Patronato Zacualpan Pro-equipamiento de Protección Civil

Aplicación web institucional y administrativa para la gestión de:

- Donadores
- Ingresos
- Egresos
- Inventario
- Transparencia pública
- Opiniones ciudadanas
- Seguimiento administrativo
- Recibos PDF
- Evidencias de equipo entregado

---

# OBJETIVO

Fortalecer a Protección Civil Zacualpan mediante un sistema transparente y organizado de participación ciudadana y recaudación comunitaria.
