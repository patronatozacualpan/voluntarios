Módulo WhatsApp

Este módulo servirá para dar seguimiento a donadores, enviar recordatorios, agradecimientos y mensajes informativos del patronato.

Por ahora funcionará como seguimiento manual mediante enlaces WhatsApp. En una fase posterior se conectará con WhatsApp Cloud API y plantillas aprobadas por Meta.

Próximas funciones

  • Recordatorio manual por donador.
  • Agradecimiento automático después de registrar ingreso.
  • Plantillas de cobranza amable.
  • Mensajes para donadores atrasados.
  • Registro de historial de mensajes.
