Registrar egreso

Este módulo registra gastos del patronato con comprobante obligatorio. Cada egreso queda guardado para transparencia y control administrativo.

Reglas del módulo

  • Solo la tesorera puede registrar egresos.
  • El monto debe ser mayor a cero.
  • Todo egreso debe llevar comprobante.
  • El comprobante puede ser imagen o PDF.
  • No se deben borrar egresos desde el panel.
